# Systems Engineering Lab

A personal repository for exploring **core concepts across modern computing systems** through small, focused, hands-on projects.
The goal of this lab is to build a strong **engineering foundation** by implementing practical examples in areas such as **networking, cloud infrastructure, artificial intelligence, and security**.

---

## Purpose

This repository acts as a **technical sandbox** for learning and experimentation.
Instead of only studying theory, each concept is implemented through **small practical projects** that demonstrate how systems actually work.

The focus areas include:

* **Networking & Distributed Systems**
* **Cloud & Infrastructure**
* **Artificial Intelligence / Machine Learning**
* **Security & Protocol Analysis**
* **Systems Programming Concepts**


## Philosophy

The lab follows a **learn-by-building approach**:

1. Study the core concept
2. Implement a minimal working version
3. Analyze how the system behaves
4. Document insights and improvements

This approach helps deepen understanding of **how real systems are built and interact**.
